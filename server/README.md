# Meu Espresso - Server

An e-commerce project for coffee sales ☕🛒

## 📦 Install

- Install dependencies.

```bash
pnpm i
```

## 🚀 Quick Start

- Run server.

```bash
pnpm dev
```
- Run tests.

```bash
pnpm tests
```
- Run tests with coverage.

```bash
pnpm tests:cov
```

## 📚 Anotations

Server hosted in [http://localhost:3333/](http://localhost:3333/).
